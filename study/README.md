## 📖 学习建议

### 每个章节的学习模式

每个章节（2 小时）建议按以下节奏：

1. **理论学习（30-40 分钟）**
   - 阅读章节文档
   - 理解核心概念和设计思路
   - 记录关键知识点

2. **代码阅读（40-50 分钟）**
   - 打开 VS Code，按指引阅读源码
   - 添加注释和理解笔记
   - 画出关键流程图

3. **实践操作（30-40 分钟）**
   - 运行章节中的示例  
   - 完成练习任务
   - 验证理解效果

### 通关技巧

✅ **每天坚持 2 小时**：建议固定时间学习，保持连续性  
✅ **做笔记**：在代码中添加注释，建立自己的理解体系  
✅ **多实验**：不要只看代码，要改、要跑、要调试  
✅ **完成练习**：每章的练习是检验理解的最好方式  
✅ **通过里程碑**：认真完成每个里程碑测试，查漏补缺

### 工具准备

- **IDE**：VS Code + Python 插件
- **终端**：PowerShell / Git Bash
- **笔记工具**：Notion / Obsidian / Markdown 编辑器
- **画图工具**：draw.io / Excalidraw（画架构图）
